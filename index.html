MicroBit Key Mapper

View on GitHub Download .zip Download .tar.gz

Mapping the A and B buttons on the MicroBit

Mappings - Arrow

A button = Up arrow

B button = Down arrow

A + B button = Right Contorl

Mappings - Letters

A button = A

B button = B

A + B button = c

How to run/edit

  1. Download it or use git 
    git clone https://github.com/SGlombicki/MicroBit-A-B-Key-mapped
  2. Install dependencies
pip install -r requirements.txt
  1. Run it 
    Python Setup.py
